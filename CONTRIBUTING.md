# دليل المساهمة

شكراً لاهتمامك بالمساهمة في مشروع Google Index Checker!

## كيفية المساهمة

### الإبلاغ عن أخطاء
1. تأكد من أن الخطأ لم يتم الإبلاغ عنه مسبقاً
2. استخدم قالب الإبلاغ عن الأخطاء
3. ارفق خطوات إعادة إنتاج الخطأ

### اقتراح ميزات جديدة
1. اشرح الفكرة بوضوح
2. اذكر فوائدها للمستخدمين
3. قدم أمثلة إن أمكن

### إرسال Pull Request
1. Fork المشروع
2. أنشئ فرعاً جديداً
3. أضف تغييراتك
4. ارفع التغييرات
5. افتح Pull Request

## معايير الكود
- استخدم تنسيق الكود الموجود
- اكتب تعليقات واضحة
- اختبر تغييراتك قبل الإرسال
